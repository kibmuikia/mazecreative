# mazecreative

A graphic design website that will showcase services of the company Maze Media Creative.

## Project setup

```
yarn install
```

or

```
npm install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

or

```
npm run serve
```

### Compiles and minifies for production

```
yarn run build
```

or

```
npm run build
```

### Run your tests

```
yarn run test
```

### Lints and fixes files

```
yarn run lint
```
